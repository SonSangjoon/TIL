# 좋은 알고리즘의 조건
- 적절한 입력
- 적절한 출력
- 명확성
- 유한성
- 효율성

# 자료구조와 알고리즘의 관계
- 자료구조 -> 알고리즘 : 자료구조의 활용을 통해 문제를 해결
- 알고리즘 -> 자료구조 : 자료구조를 구현하기 위해 사용

# 시간 복잡도와 공간 복잡도
- 시간복잡도: 코드의 실핸 속도
- 공간복잡도: 메모리 사용량

# 배열의 시간 복잡도 / 공간 복잡도 
배열의 시간 복잡도
- 인덱스틀 알때 : O(1)
- 인덱스를 모를 때 : 하나씩 검사할 경우 O(N)
- 배열 전체 순회 : O(N)
- 자료끝에 추가 : O(1)
- 자료 중간에 추가 : O(N)

배열의 공간 복잡도 = O(N)

# 배열 vs 해쉬 (trade-off 관계)
- 해쉬 : 식별자가 있는 데이터 (시간 복잡도 down / 공간복잡도 up)
- 배욜 : 식별자가 없는 데이터 (시간 복잡도 up / 공간복잡도 down)
